### Changed

- Standard onboarding now defaults to a minimal solo-dev setup. Hooks, changelog
  ceremony, PR templates, CODEOWNERS, Dependabot, GitHub labels, branch
  protection, runner-backed workflows, agent lifecycle scripts,
  doc-health/doc-sweep runners, and template libraries remain available but are
  explicit opt-ins. Headless audit now reports whether startup readiness is
  evaluating the minimal or full process profile.
